# Hangman Game (Python)

A simple and classic **Hangman** game built with Python.  
You try to guess the hidden word letter by letter, and you have **6 attempts** before the hangman is fully drawn and the game ends.

## Features
- Guess letters to reveal a hidden word
- 6 attempts before losing
- Input validation for repeated letters
- Clear terminal-based interface
- Easy to edit and customize word lists

## How to Run
```bash
python hangman.py
